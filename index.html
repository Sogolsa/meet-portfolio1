Meet App

The Meet App is a progressive web app with the ability to work offline and with a serverless backend, developed using a test-driven development (TDD) approach , where the tests have been written before the actual functionality of the app in code.

The Meet App is designed to provide users with seamless experience in exploring upcoming events in different cities, and uses Google Calendar API to fetch upcoming events.

PWA utilizes service workers and web app manifest so that the user can use the app offline, and be able to install the app (acts as a native app).

Using Recharts library, a pie chart and a scatter plot has been implemented in the app for a better user experience.

Tools, Skills, Methodologies:

React, Google Calendar API, AWS Lambda(serverless functions), Test-Driven Development (TDD), OAuth2 Authentication, git and github, data visualization libraries, atatus for monitoring performance
